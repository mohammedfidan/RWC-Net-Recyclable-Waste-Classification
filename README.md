# 🗑️ RWC-Net: Recyclable Waste Classification System

A deep learning-based waste classification system built using YOLO, 
designed to automate and improve recycling efficiency.

## 📌 Overview
RWC-Net is a robust deep learning model that classifies recyclable 
waste using YOLO integrated with CNN. It uses Score-CAM to 
visualize the model's focus areas, making it interpretable 
and reliable.

## 🎯 Objectives
- Develop RWC-Net using YOLO for accurate waste classification
- Achieve high accuracy on a predefined waste dataset
- Use Score-CAM for explainability
- Deploy a scalable web + mobile solution
