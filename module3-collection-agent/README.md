# Module 3 - Collection Agent Panel

## 📌 Overview
The Collection Agent module of the RWC-Net system provides a 
dedicated interface for waste collection agents to manage and 
track recyclable waste pickup activities in their assigned areas.

## 🔑 Features
- Secure agent login and authentication
- View assigned waste collection tasks
- Update waste collection status in real-time
- View classified waste categories from RWC-Net model
- Track collection history and reports
- Coordinate with admin for task management
