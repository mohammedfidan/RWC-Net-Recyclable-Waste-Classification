# Module 1 - Admin Panel

## 📌 Overview
The Admin module of the RWC-Net system provides a centralized 
dashboard for managing the recyclable waste classification platform. 
Admins can monitor agents, users, and waste classification activities.

## 🔑 Features
- Secure admin login and authentication
- Dashboard to monitor overall system activity
- Manage collection agents and user accounts
- View waste classification reports and statistics
- Real-time updates on waste collection status
